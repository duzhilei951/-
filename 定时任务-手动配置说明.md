# 定时任务手动配置说明

由于权限和编码问题，建议手动配置定时任务。

## 配置步骤

### 第一步：以管理员身份打开命令提示符

1. 按 `Win + X`
2. 选择 **"终端(管理员)"** 或 **"Windows PowerShell (管理员)"**
3. 点击"是"允许管理员权限

### 第二步：执行创建命令

复制粘贴以下命令：

```cmd
schtasks /create /tn "小远智-AI模型每日监测" /tr "PowerShell -ExecutionPolicy Bypass -File C:\Users\远智教育\.openclaw\workspace\小远智\scripts\run-daily.ps1" /sc daily /st 09:00 /f
```

### 第三步：验证创建成功

```cmd
schtasks /query /tn "小远智-AI模型每日监测"
```

应该显示：
- 任务名：\小远智-AI模型每日监测
- 下次运行时间：明天 9:00:00

## 测试运行

立即运行一次测试：

```cmd
schtasks /run /tn "小远智-AI模型每日监测"
```

查看日志：

```cmd
type "C:\Users\远智教育\.openclaw\workspace\小远智\logs\daily_2026-03-01.log"
```

## 管理命令

```cmd
# 查看状态
schtasks /query /tn "小远智-AI模型每日监测" /v

# 暂停
schtasks /change /tn "小远智-AI模型每日监测" /disable

# 恢复
schtasks /change /tn "小远智-AI模型每日监测" /enable

# 删除
schtasks /delete /tn "小远智-AI模型每日监测" /f
```

## 脚本说明

使用的新脚本：`run-daily.ps1`

特点：
- 简化版 PowerShell 脚本
- 避免中文路径问题
- 详细的日志记录
- 自动处理错误

---

*配置完成后，系统将在每天上午9:00自动运行*
